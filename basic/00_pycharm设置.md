### shortcut
```bash
# Pycharm快捷键设置
File - Settings - Keymap
# 代码提示部分
m/f/v/c/p  # 方法/字段/变量/类/p参数
# 设置
Ctrl + Alt + S
# 代码格式化
Ctrl + Alt + L
# 查看函数详细注释
F2
# 取消sql语句高亮背景
File - Settings - Editor - Language Injections - 去掉勾选python: "SQL select/delete/insert/update/create"
```
